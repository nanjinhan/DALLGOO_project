# 이미지 / 영상 넣는 곳 (달구 회사소개 랜딩)

여기(`frontend/public/images/`)에 아래 파일명으로 넣으면 랜딩페이지에 자동 표시됩니다.
**영상이 있으면 영상(자동재생·무한반복·음소거)을 먼저 쓰고, 없으면 이미지, 둘 다 없으면 자리표시**가 보입니다.

| 위치 / 용도 | 영상(우선) | 이미지(대체·포스터) | 권장 비율·크기 |
|---|---|---|---|
| 히어로(첫 화면) | `hero.mp4` | `hero.jpg` | 21:9 가로형, 1920×823↑ |
| "치료적 외출" 사회공헌 | `outing.mp4` | `outing.jpg` | 4:3 가로형, 1200×900↑ |
| 상단 로고(선택) | — | `logo.svg` | 높이 28px 기준 |

## 영상 팁
- 형식: **mp4(H.264)** 권장. webm도 가능하나 파일명을 코드와 맞춰야 함.
- 브라우저 자동재생 정책상 **자동 음소거**로 재생됩니다(소리 X). 배경 영상용으로 적합.
- 무한 반복되므로 **5~15초 짧은 루프 영상**이 좋습니다.
- 용량이 크면 첫 로딩이 느려지니 가급적 **10MB 이하**로 압축하세요.
- `hero.mp4`와 `hero.jpg`를 둘 다 두면, 영상 로딩 전/실패 시 이미지가 포스터로 보입니다.

## 공통
- 정적 파일이라 빌드 없이 바로 반영됩니다. (`/images/hero.mp4` 경로로 서빙)
- 다른 이름·다른 위치에 더 넣고 싶으면 말씀해주세요. 슬롯을 추가해드립니다.
